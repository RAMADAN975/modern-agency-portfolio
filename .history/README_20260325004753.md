# Agency.AI - Digital Agency Portfolio

A modern, responsive digital agency landing page built with **React** and **Bootstrap 5**.

## 🚀 Features
- **Dark/Light Mode**: Seamless theme switching.
- **Modern UI**: Clean design using the Manrope font.
- **Animations**: Integrated with AOS (Animate On Scroll) for a dynamic experience.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Contact Form**: Functional frontend integration.

## 🛠️ Tech Stack
- React.js
- Bootstrap 5
- AOS (Animate On Scroll)
- Lucide React (Icons)

## 📖 How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
